Projet 1 : Gestion de ligue

Voir le projet

Le projet consiste à développer une application en ligne de commande, actuellement mono-utilisateur, que nous souhaitons améliorer pour la rendre multi-utilisateurs en intégrant une base de données. L'objectif principal est de permettre une gestion plus efficace des employés au sein de différentes ligues, en désignant un administrateur par ligue. Cet administrateur aura pour mission de recenser et gérer les employés de sa ligue via une application bureau.

L'application propose différents niveaux d'habilitation pour les utilisateurs :

Employé de ligue : Un employé standard peut accéder à l'application pour consulter les informations comme un annuaire, mais il ne dispose d'aucun droit d'écriture.

Administrateur de ligue : Un employé désigné comme administrateur de sa ligue dispose de droits d'écriture. Il peut gérer la liste des employés de sa propre ligue via une application bureau.

Super-administrateur : Ce niveau d'utilisateur a un accès en écriture à tous les employés de toutes les ligues. Il peut gérer les comptes des administrateurs via une application en ligne de commande.

Projet 2 : Intranet Maison des ligues

Voir le projet

Le projet "Maison des ligues" est une application web dynamique, conçue pour faciliter la gestion et la communication au sein d'une organisation. Développée avec les technologies web standards HTML, CSS et enrichie par la logique serveur de PHP et l'interactivité de JavaScript, cette plateforme offre une expérience utilisateur à la fois intuitive et sécurisée, avec des fonctionnalités adaptées à deux types d'utilisateurs : les collaborateurs (utilisateurs standards) et les administrateurs.

Les fonctionnalités clés proposées par cette application sont les suivantes :

A. Utilisateur standard (collaborateur) : Accès sécurisé : L'application est protégée par un système d'authentification robuste nécessitant un identifiant et un mot de passe. Accueil personnalisé : Après connexion, l'utilisateur découvre une page d'accueil présentant un collaborateur aléatoire, avec la possibilité d'explorer d'autres profils. Annuaire des collaborateurs : Une section dédiée liste l'ensemble des collaborateurs de l'entreprise sous forme de fiches informatives, offrant des options de filtrage par nom, localisation et catégorie. Gestion du profil personnel : Chaque utilisateur a la possibilité de modifier ses informations personnelles, y compris ses identifiants de connexion, via une page accessible depuis son avatar. Déconnexion sécurisée : Un mécanisme de déconnexion permet à l'utilisateur de quitter l'application en toute sécurité, bloquant l'accès aux pages protégées jusqu'à la prochaine connexion.

B. Administrateur : Fonctionnalités étendues : L'administrateur bénéficie de toutes les fonctionnalités de l'utilisateur standard, auxquelles s'ajoutent des droits spécifiques pour la gestion des collaborateurs. Gestion complète des collaborateurs : L'administrateur a la capacité d'ajouter de nouveaux collaborateurs, de modifier les informations existantes et de supprimer des profils. Formulaire d'ajout simplifié : Une page dédiée offre une interface conviviale pour la création rapide de nouveaux profils de collaborateurs. Attribution du rôle d'administrateur : L'administrateur a également la possibilité de promouvoir d'autres utilisateurs au rôle d'administrateur lors de la modification de leurs informations.

Projet 3 : Faux Gmail

Voir le projet

Le projet "Faux Gmail" est une interface de messagerie web minimaliste, construite avec les fondations du web : HTML pour sa structure et CSS pour son apparence. Son principe fondamental est de démontrer la facilité avec laquelle une fausse page internet, imitant des interfaces familières, peut être créée à des fins potentiellement malveillantes.

Bien que visuellement inspirée des plateformes de messagerie courantes, "Faux Gmail" n'est pas une véritable application de messagerie. Son objectif premier est de souligner la simplicité technique requise pour concevoir une page web trompeuse, capable de leurrer des utilisateurs peu méfiants.

En se concentrant sur une apparence familière et des interactions de base, cette interface met en évidence comment des acteurs malintentionnés pourraient exploiter la confiance des utilisateurs envers des designs connus pour collecter des informations sensibles ou mener d'autres actions malveillantes.

"Faux Gmail", réalisé uniquement en HTML et CSS, illustre de manière concrète que la sophistication visuelle d'une page web ne garantit pas son authenticité ou sa sécurité. Ce projet se veut une piqûre de rappel sur la nécessité de la vigilance et de la sensibilisation aux risques de phishing et d'ingénierie sociale sur internet. Il prouve qu'avec des compétences techniques de base, il est aisé de créer une contrefaçon d'interface web suffisamment convaincante pour tromper.

"Compteur" est une simple application web (HTML, CSS, JavaScript) pour les joueurs d'Elden Ring. Elle permet de compter vos décès via des boutons d'ajout et de retrait, avec un affichage du total. Ce petit projet illustre l'interactivité web basique et rend hommage à la difficulté du jeu.
