Projet 1 : Gestion de ligue

Voir le projet

Le projet consiste à développer une application en ligne de commande, actuellement mono-utilisateur, que nous souhaitons améliorer pour la rendre multi-utilisateurs en intégrant une base de données. L'objectif principal est de permettre une gestion plus efficace des employés au sein de différentes ligues, en désignant un administrateur par ligue. Cet administrateur aura pour mission de recenser et gérer les employés de sa ligue via une application bureau. L'application propose différents niveaux d'habilitation pour les utilisateurs : Employé de ligue : Un employé standard peut accéder à l'application pour consulter les informations comme un annuaire, mais il ne dispose d'aucun droit d'écriture. Administrateur de ligue : Un employé désigné comme administrateur de sa ligue dispose de droits d'écriture. Il peut gérer la liste des employés de sa propre ligue via une application bureau. Super-administrateur : Ce niveau d'utilisateur a un accès en écriture à tous les employés de toutes les ligues. Il peut gérer les comptes des administrateurs via une application en ligne de commande.
